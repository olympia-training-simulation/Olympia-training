yêu cầu:
    Node.js: Phiên bản mới nhất (Sẽ bao gồm trong file cài đặt)
    Browser: cập nhật phiên bản mới nhất 
    * Chrome/Firefox/CocCoc/...





Sửa dữ liệu và câu hỏi:

Để thay đổi dữ liệu và câu hỏi, truy cập vào trang web : “http://nolympia.tech/xuatde/”
sau khi hoàn thành website sẽ xuất ra một file:”data.json”. Người quản lý thay thế file cũ ở trong thư mục “data”. Sau đó bấm nút :”khởi chạy lại server” để xác nhận những thay đổi. 



Giao diện điều khiển của người dùng:
    Install: npm install
    Run: npm start
    Debug: npm run debug

    Reset data: npm run set-data default
    Set data from file: npm run set-data <./path/to/file>


The server's IP will appear in command line when you run it.
